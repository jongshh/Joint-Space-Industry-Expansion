## :rocket: About JSIE Pack
**Joint Space Industry Expedition (JSIE)** introduces brand new cities and bases for Kerbals looking for experiences that have never existed before. 

Expand your horizons and launch your missions from entirely new locations.

**Now, Let's Explore!**

## :package: Installation

### :warning: Requirements
To run JSIE correctly, you **must** install the following mods and packs:

* **Kerbal Konstructs** (and all its dependencies)
* **Required KK Packs:** [Check this Sheet](https://docs.google.com/spreadsheets/d/1VZ0i9FgXNZKLybHH4LQEu_6AzNUXhfGn1atKMdY7ZOI/edit?gid=321312601#gid=321312601)

> **Recommendation:** We highly recommend using **CKAN** to install Kerbal Konstructs to ensure all dependencies are handled automatically.

## :package: Installation
**Recommended:** We highly recommend using **CKAN** to install this mod and its dependencies.

If installing manually:
1. Ensure all dependent mods and packs are installed.
2. Always follow the included instructions carefully.
